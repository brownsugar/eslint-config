# ESLint Config

Preferred ESLint configs for myself.

## For TypeScript

### Install

```bash
yarn add -D @brownsugar/eslint-config-typescript
```

### .eslintrc

```json
{
  "root": true,
  "extends": [
    "@brownsugar/typescript"
  ]
}
```
